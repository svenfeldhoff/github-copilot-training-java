# Module 3 - Version Control and Quality

## Goal
Use Copilot while preserving clean commit quality.

## Exercise

1. Ask Copilot to draft a commit message for your feature.
2. Review the diff and request a code review summary from Copilot.
3. Split changes into two commits: feature and tests.

## Quality Gates

- All tests pass before commit.
- Commit messages explain why, not just what.
- No unrelated formatting changes.
