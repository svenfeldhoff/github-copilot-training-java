# Module 6 - Vibe Coding Challenge

## Goal
Use Copilot to build a complete feature quickly and then harden it.

## Challenge

Create a minimal dashboard endpoint set:

- `GET /api/reports/summary`
- return total tasks, done tasks, and completion rate

## Constraints

- Keep code simple and readable.
- Add tests first or immediately after generation.
- Refactor one generated section for clarity.

## Debrief

- What parts did Copilot accelerate most?
- Where did manual engineering judgment matter most?
- Verify `GET /api/reports/summary` is visible and returns the correct schema at [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html).
